# Преобразование Фурье и сингулярно-спектральное разложение (SSA) пульсового сигнала
Определить и сравнить частоты главных компонент Фурье-разложения с частотами основных гармоник сингулярно-спектрального разложения (SSA) на выборке пульсовых сигналов.
Вход: Выборка пульсовых сигналов объема N. Частота дискретизации сигнала 100 Гц.
Двухбайтовая кодировка.
Выход:
- N значений частоты главной компоненты Фурье-разложения
- N значений частоты главной гармоники сингулярно-спектрального разложения (SSA)
- График их зависимости: абсцисса – частота по Фурье, ордината – частота по SSA,
линейный тренд – биссектриса первой координатной четверти

## 1. Преобразование Фурье

Преобразование Фурье позволяет представить временной сигнал как сумму синусоидальных колебаний различных частот.

Для дискретного сигнала длины `N` дискретное преобразование Фурье (DFT) определяется формулой:

\[
X_k = \sum_{n=0}^{N-1} x_n e^{-i 2\pi k n / N}, \quad k = 0,1,...,N-1
\]

где:

- `x_n` — значение сигнала во времени  
- `X_k` — спектральный коэффициент  
- `N` — количество отсчётов сигнала  

Частота соответствующей гармоники определяется как:

\[
f_k = \frac{k \cdot f_s}{N}
\]

где:

- `f_s` — частота дискретизации  
- `k` — индекс спектральной компоненты  

Главная частота сигнала определяется как частота компоненты с максимальной амплитудой спектра:

\[
f_{main} = f_{argmax(|X_k|)}
\]

На практике используется быстрое преобразование Фурье (FFT), реализованное в библиотеке `numpy`.

## 2. Сингулярно-спектральный анализ (SSA)

Сингулярно-спектральный анализ — метод анализа временных рядов, основанный на сингулярном разложении матриц (SVD).

Метод позволяет разделить сигнал на:

- тренд
- периодические компоненты
- шум

### 2.1 Построение траекторной матрицы

Для временного ряда `x1, x2, ..., xN` выбирается окно длины `L`.

Формируется **траекторная (Hankel) матрица**:

\[
X =
\begin{bmatrix}
x_1 & x_2 & x_3 & \dots & x_K \\
x_2 & x_3 & x_4 & \dots & x_{K+1} \\
x_3 & x_4 & x_5 & \dots & x_{K+2} \\
\vdots & \vdots & \vdots & \ddots & \vdots \\
x_L & x_{L+1} & x_{L+2} & \dots & x_N
\end{bmatrix}
\]

где:

\[
K = N - L + 1
\]

---

### 2.2 Сингулярное разложение

К матрице применяется SVD:

\[
X = U \Sigma V^T
\]

где:

- `U` — матрица собственных векторов  
- `Σ` — диагональная матрица сингулярных значений  
- `V` — матрица коэффициентов  

Каждое сингулярное значение соответствует компоненте сигнала.

---

### 2.3 Реконструкция компонент

Каждая компонента восстанавливается как:

\[
X_i = \sigma_i U_i V_i^T
\]

После этого выполняется **диагональное усреднение** (Hankelization), позволяющее получить временной ряд компоненты.

---

### 2.4 Определение основной гармоники SSA

1. Берутся первые несколько компонент SSA
2. Для каждой вычисляется FFT
3. Определяется частота максимального спектрального пика

Главная частота SSA:

\[
f_{SSA} = argmax(|FFT(RC_i)|)
\]

где `RC_i` — восстановленная компонента.

# 3. Чтение входного сигнала

Сигнал хранится в CSV: колонка 0 - время, колонка 1 - PPG1. Перед анализом - срез 10-110 с и фильтр 0.5-5 Гц.

# 4. Итоговый алгоритм

1. Считать `N` сигналов из файлов.

2. Для каждого сигнала вычислить:

   * частоту главной компоненты **Фурье**
   * частоту главной гармоники **SSA**

3. Сформировать два массива частот:

```
X — частоты, полученные методом Фурье  
Y — частоты, полученные методом SSA
```

4. Построить график зависимости частот.

5. Добавить на график линию

[
y = x
]

которая является **биссектрисой первой координатной четверти** и показывает идеальное совпадение результатов методов.

# 11. Ожидаемый результат

После выполнения программы должны быть получены:

* массив из `N` значений частоты главной компоненты **Фурье**
* массив из `N` значений частоты главной гармоники **SSA**
* график зависимости

```
X — частота Фурье
Y — частота SSA
```

Если оба метода определяют одну и ту же основную частоту сигнала, точки на графике будут располагаться вдоль линии

[
f_{SSA} = f_{Fourier}
]
